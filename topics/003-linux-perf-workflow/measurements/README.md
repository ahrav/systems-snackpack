# Measurements

Each note records the CPU, operating system, toolchain, build flags, input,
warm-up count, trial count, statistic, and measurement boundary.

Linux PMU results must also record the kernel, `perf` version, resolved events,
event scope, CPU affinity, watchdog state, enabled time, running time, raw count,
scaled count, and any loss or throttling. Mark mechanism explanations as
inferences.
