## Boundary

probes and exact verification for the index path, or exact matching over every
document for the scan path. It excludes process startup, data generation, index
construction, warm-up, candidate reporting, and output.

for one binary, corpus, and query set. The deterministic schedule can alias host

independence assumption. It does not cover other builds, hosts, corpora, query
