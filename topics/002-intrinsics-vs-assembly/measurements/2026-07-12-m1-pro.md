# Apple M1 Pro baseline — 2026-07-12

- CPU: Apple M1 Pro
- Operating system: Darwin 25.5.0, arm64
- Toolchain: rustc 1.93.1, LLVM 21.1.8
- Command: `cargo bench -p systems-snackpack-topic-002 --bench fold`
- Input: 262,144 deterministic `u64` values (2 MiB).
- Method: the benchmark checks both paths against the reference fold, runs eight warm-up runs, alternates two implementations across 41 trials, and reports medians.

| Path | Median throughput |
|---|---:|
| Intrinsic | 22.569 GiB/s |
| Inline assembly | 22.569 GiB/s |

The benchmark measures whole-fold elapsed time. It does not measure isolated
instruction latency, identify the limiting hardware resource, or establish a
result for other CPUs, compilers, or input sizes.
