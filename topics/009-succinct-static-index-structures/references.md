# References

- [Jacobson, *Succinct Static Data Structures*](https://www.cs.cmu.edu/~sleator/papers/Jacobson-thesis.pdf) — Foundational rank/select representations and the static-query model.
- [Raman, Raman, and Rao, *Succinct Indexable Dictionaries*](https://arxiv.org/abs/0705.0552) — Compressed indexable dictionaries and the RRR block representation.
- [Vigna, *Broadword Implementation of Rank/Select Queries*](https://vigna.di.unimi.it/ftp/papers/Broadword.pdf) — Broadword rank/select implementation techniques and explicit cost boundaries.
- [Pibiri and Venturini, *Techniques for Inverted Index Compression*](https://arxiv.org/abs/2003.11835) — Elias-Fano layout, bounds, and practical query tradeoffs.
- [Vigna, *Quasi-Succinct Indices*](https://arxiv.org/abs/1206.4300) — Elias-Fano-based monotone sequence representation and search operations.
- [Claude, Navarro, and Ordonez, *The Wavelet Matrix*](https://arxiv.org/abs/1405.1220) — Wavelet-matrix layout and sequence operations.
- [Grossi, Gupta, and Vitter, *High-Order Entropy-Compressed Text Indexes*](https://ittc.ku.edu/~jsv/Papers/GGV03.entropy.pdf) — Entropy-compressed sequence indexes and compressed bitmap context.
- [Esposito, Graf, and Vigna, *RecSplit*](https://arxiv.org/abs/1910.06416) — Minimal perfect hashing with explicit space and construction tradeoffs.
- [Rust `u64::count_ones`](https://doc.rust-lang.org/std/primitive.u64.html#method.count_ones) — Language-level population-count semantics.
- [rustc `target-cpu`](https://doc.rust-lang.org/stable/rustc/codegen-options/index.html#target-cpu) — Target feature selection used for generated-code inspection.
- [Kalibera and Jones, ISMM 2013](https://doi.org/10.1145/2464157.2464163) — Process-level replication and benchmark uncertainty.
